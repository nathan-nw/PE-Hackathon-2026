Uses the Flask API POST /shorten · needs a valid user_id in the database

Local Docker: use the load balancer (NGINX), not the API container port. From the repo root run docker compose port load-balancer 80 to see the host port (often :8080 or :18080 if LB_HTTP_PORT is set in .env). A CORS error usually means the URL is not NGINX (wrong port or another process bound there). Not :5000 / not :3000 (dashboard). On Railway, use your url-shortener HTTPS URL.
