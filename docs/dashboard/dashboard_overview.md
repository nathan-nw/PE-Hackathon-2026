# Dashboard Documentation Overview

This directory contains the split documentation for the full-stack system metrics dashboard. The dashboard monitors real-time URL link throughput, IP ban controls, API load test integrations, and manages the Discord webhook alerting pipelines.

## Documentation Structure
The documentation here perfectly mirrors the source repository's nested codebase structure:

- **[Backend Engine](./backend/python_engine.md)**: Details the Python backend metrics service (`main.py`, PostgreSQL interaction, Kafka loops, and the Discord integration).
- **[Frontend Application](./src/nextjs_frontend.md)**: Details the visual Next.js interface (`app/`, component layouts, and Next configuration).
